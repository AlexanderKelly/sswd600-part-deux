Call to Action

Element

HTML

<form action="https://github.com/AlexanderKelly/sswd600-part-deux/"> <input type="submit" class="cta__github" value="Visit my GitHub"/> </form>

CSS


.cta__github{
float: center;
background: #FFFFFF;
margin-left: auto;
margin-right: auto;
color: #000000;
border: 2px solid #000000;
border-radius: 8px;
text-align: center;
text-decoration: none;
display: inline;
font-size: 32px;
}

Notes

A Call to Action is a web component that aims to encourage a user to to perform an action, such as subscribing to a websites newsletter, or viewing an affiliate sponsor's website. Examples include the subscribe button on YouTube channels, offers for a free trial of a piece of software or the option to share a news article or video on social media platforms.

On links, make sure to have a / at the end of the address, as some browsers automatically add a question mark to the end of some links.
