Text created under the h1tag is typically used to create main headings for a page. It should have the largest font

Header2

Text created under the h2 tag is typically used to for sub-headings for a page. It should have the second-largest font, but this can change based on needs of the developer

Further h tags can be employed to denote a descending hierarchy of headings in a page
