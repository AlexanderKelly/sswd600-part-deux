Navigation

Element

HTML

<button type="button" class="button__next">Next <strong>></strong></button> <button type="button" class="button__previous"> <strong><</strong>Previous</button>

CSS


.button__next{
background: #3873BF;
color: #FFFFFF;
border: 2px solid #FFFFFF;
border-radius: 8px;
text-align: center;
text-decoration: none;
display: inline;
font-size: 24px;
}
.button__next:hover{
border: 2px solid #3873BF;
}
.button__previous{
background: #3873BF;
color: #FFFFFF;
border: 2px solid #FFFFFF;
border-radius: 8px;
text-align: center;
text-decoration: none;
display: inline;
font-size: 24px;
}
.button__previous:hover{
border: 2px solid #3873BF;
}

Notes

These two buttons are templates for navigation buttons to be incorporated into the website. Their purpose is to help facilitate navigation through news articles or slide shows on the website. In the first case, they would be located both at the top and bottom of an article, and in the latter would be positioned either side of the slides, optimally not covering any of the slide so as to not cover any content.

Social Media

Element

HTML

<form action="https://twitter.com/I_am_AlexKelly/"> <input type="submit" class="button__twitter" value="Twitter"/> </form> <form action="https://github.com/AlexanderKelly/"> <input type="submit" class="button__github" value="GitHub"/> </form>

CSS


.button__twitter{
background: #FFFFFF;
color: #00ACED;
border: 2px solid #00ACED;
border-radius: 8px;
text-align: center;
text-decoration: none;
display: inline;
font-size: 24px;
}
.button__github{
background: #FFFFFF;
color: #000000;
border: 2px solid #000000;
border-radius: 8px;
text-align: center;
text-decoration: none;
display: inline;
font-size: 24px;
}

Notes

These buttons serve as links to external social media sites such as Facebook, Twitter, Google+ or LinkedIn. They are comprised of the given name for these social media companies with a light grey background and a border that matches a colour present in the original logo. Alternatively, use a button that these sites have developed themselves.
